## Getting Started
```bash
cd todoapp-using-server-actions
npx create-next-app@latest .
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Dependencies
```bash
npm install sequelize mysql2 pg-hstore moment-timezone
```

## .env File
```bash
DB_NAME = 
DB_USER = 
DB_PASSWORD = 
DB_HOST = 
```
